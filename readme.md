# SimCompare

A tool to display all current Drillisch tariffs and compare them

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Usage

```bash
python src/main.py
```

---

## Parameters

| Name      | Type | Description                                |
|-----------|------|--------------------------------------------|
| `--clean` | ``   | Starts a completely new crawl of all sites |
